# The Compiler

You can download the latest version of the compiler [here](./getting_started.md).

## Init Project

```
ckc init
```

## Build Project

```
ckc build
```

## Watch Project

```
ckc watch
```

## Other directory

If you are not in the right directory you can add a path:

```
ckc <command> -p ./Somewhere
```

`<command>` is `build` or `watch`.
